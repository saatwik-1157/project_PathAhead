The one rule

Prompt → test → iterate

Prompting is an experimental skill, not a magic-words game. Keep a prompt journal: write a prompt, score the output, change one thing, run it again. Every project below ships with the prompts that power it.

Important: Prompt engineering is strongest as a multiplier: pair it with a domain you care about — coding, content, data, research — and you become the person who gets 10× more out of AI than everyone around you.
