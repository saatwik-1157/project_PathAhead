The Roadmap

Four phases, step by step

0

Foundations

Weeks 1–5

The Linux, Git, and scripting base everything sits on.

Week 1

Linux & CLI

Shell, files, permissions, processes.

Week 2

Networking basics

DNS, HTTP, ports, load balancing.

Week 3

Git workflows

Branches, PRs, merge strategies.

Week 4

Scripting

Bash and Python for automation.

Week 5

Programming refresher

Enough code to build tooling.

1

Containers & CI/CD

Weeks 6–11

Package apps and ship them automatically.

Week 6

Docker

Images, containers, Dockerfiles.

Week 7

Docker Compose

Multi-service local environments.

Week 8

CI/CD concepts

Pipelines with GitHub Actions.

Week 9

Build & deploy

Automated test-and-deploy workflows.

Week 10

Registries

Push and pull container images.

Week 11 · Project ▶

Containerized pipeline

Dockerize an app and auto-deploy it.

2

Cloud & Orchestration

Weeks 12–17

Run workloads on AWS and Kubernetes.

Week 12

Cloud fundamentals

AWS core: EC2, S3, IAM.

Week 13

Cloud networking

VPCs, subnets, security groups.

Week 14

Kubernetes basics

Pods, deployments, services.

Week 15

K8s scaling

Replicas, autoscaling, config.

Week 16

Infrastructure as Code

Terraform to define infrastructure.

Week 17 · Project ▶

Deploy to Kubernetes

Run your app on a K8s cluster.

3

Monitoring & SRE

Weeks 18–22

Keep production healthy and secure.

Week 18

Monitoring

Prometheus and Grafana dashboards.

Week 19

Logging

Centralized logs with the ELK stack.

Week 20

Security & secrets

Secrets management, least privilege.

Week 21

Cert prep

AWS Cloud Practitioner study.

Week 22 · Capstone ▶

Full platform

CI/CD + IaC + K8s + monitoring, end to end.
