The Roadmap

Four phases, step by step

0

Foundations

Weeks 1–4

Structure and style the web — HTML, CSS, and Git.

Week 1

HTML fundamentals

Semantic tags, forms, links, images, accessibility.

Week 2

CSS & layout

The box model, Flexbox, and CSS Grid.

Week 3

Responsive design

Media queries, variables, transitions, mobile-first.

Week 4

Git & tooling

Git, GitHub, VS Code; deploy a static site free.

1

JavaScript

Weeks 5–9

Make pages interactive with real JavaScript.

Week 5

JS basics

Variables, functions, arrays, objects, loops.

Week 6

DOM & events

Manipulate the page, handle clicks and input.

Week 7

Async & APIs

Promises, fetch, JSON, calling real APIs.

Week 8

Modern JS

ES6+, modules, npm, array methods.

Week 9 · Project ▶

Interactive app

Build a weather or to-do app using an API.

2

Frontend with React

Weeks 10–15

Build modern single-page apps with React.

Week 10

React basics

Components, JSX, props, rendering.

Week 11

State & hooks

useState, useEffect, lifting state.

Week 12

Routing & forms

React Router, controlled inputs, validation.

Week 13

Data fetching

Call APIs, handle loading and error states.

Week 14

Styling & UI

Tailwind or a component library.

Week 15 · Project ▶

React SPA

Ship a movie or product browser app.

3

Backend & Full-Stack

Weeks 16–22

Add a server, a database, and deploy the whole thing.

Week 16

Node & Express

Servers, routes, building REST APIs.

Week 17

Databases

SQL vs NoSQL; MongoDB or PostgreSQL.

Week 18

CRUD & auth

JWT, sessions, password hashing.

Week 19

Full-stack wiring

Connect the React front end to your API.

Week 20

Deployment

Vercel/Render, env vars, basic CI.

Week 21

Testing & polish

Error handling, validation, cleanup.

Week 22 · Capstone ▶

Full-stack app

Ship a complete deployed MERN/PERN app.
